# UI Implementation

Take a design review or mockup and output a React component.

1. Use only existing DUI primitives such as Button, Input, and Card.
2. Do not define ad-hoc CSS classes or separate stylesheet blocks.
3. Use logical properties for RTL compatibility, such as `ms-*` and `pe-*`.
4. Make the layout responsive for mobile, tablet, and desktop views.
